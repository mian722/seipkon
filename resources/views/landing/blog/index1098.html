5 Great New Features on Offerslook

Dec 12, 2016 By Offerslook

After being “silent” for quite some time (that’s because our tech team were developing intensely for a big upgrade), today we released a bunch of new features! Check how these features can make your job a lot easier.

Server Log

The server logs report is a summary of all problematic traffic, and helps you to discover conversion problems, tracking errors and also detect fraudulent activities. These activities and problems may include, users not meeting geo-targeting, affiliates mis-formatting a tracking link, affiliates delivering conversions over caps, etc.

Check out this tutorial.

Advertiser Invoices

Offerslook now allows you to store advertisers’ financial information into the system, and you can also generate invoices to advertisers manually or automatically.

Check how to add advertiser financial information and how to generate advertiser invoices.

Affiliate Invoices

Similar to advertiser invoices, affiliate invoice features allow you to store affiliates’ financial information and generate invoices manually or automatically.

Check how to add affiliate financial information and how to generate affiliate invoices.

Testing Affiliate Postback

Testing affiliate postback was already there in Offerslook, but we have optimized the testing process and also revised the geeky wordings. Now you’ll be able to test postback much easier.

Check out how to test affiliate postback.

Expand Advertiser IP’s

Previously some partners find it not enough to list all advertiser IP’s, and now we expand this setting. You’ll be able to set 50 combinations of advertiser IP’s.

Check out a more detail tutorial.

Coming Next…

Although it took us some time to have this product upgrade, we have optimized our product upgrade structure, and in the coming future you’ll expect faster product upgrades. And we strong believe you’ll benefit from these new features, to make your work a lot easier.
