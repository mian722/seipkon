借助大数据之力，打造全球广告效果营销网络平台：Offerslook 受邀参加AWSome Day

May 27, 2018 By Offerslook

5月24日，由AWS举办的 AWSome Day 广州站在广州粤海喜来登酒店举行。作为行业领先的教育与技术新知的云计算研讨会， AWSome Day吸引了500多位IT领域技术及业务决策者参会。AWS特别邀请Offerslook的技术总监唐恒昆先生，作为先锋客户代表做了《Offerslook借助大数据之力，打造全球广告效果营销网络平台》的精彩演讲。

 

作为全球领先的效果营销管理云平台，offerslook通过云计算，大数据，智能算法等创新技术手段，旨在帮助全球营销者提供更稳定，更高效，更智能的效果广告追踪平台。经过2年的发展，offerslook用户已经遍布全球40多个国家。

 

用户每年以几十倍速度增长
2年，点击量完成从300w到5个亿的激增过程
每天新增数据量为TB级别
每分钟动态请求达百万次
…

如此迅猛的业务发展和庞大的数据请求，在技术架构上Offerslook是如何做到的？

 

 

在现场，offerslook技术总监唐恒昆揭开了秘密，向与会者分享了offerslook系统架构的设计与实践，重点详细阐述了如何结合AWS的S3 Select功能和自身技术优势，快速助力Offerslook全球业务快速发展。唐总总结了在Offerslook高速业务发展环境下架构设计中最为突出的三大挑战：数据巨大、响应时间要求极短，突增请求量频发。为保障为客户提供更快更稳定的平台和服务，Offerslook遵循创新设计、优化性能、便捷运维这几个原则来进行架构设计。Amazon S3 Select的应用正是创新设计的诠释。

 

Amazon S3 Select 是一个全新的 Amazon S3 功能，借助 S3 Select，应用程序可以将过滤和访问对象内部数据的繁重工作卸载到 Amazon S3 服务。通过减少必须由应用程序加载和处理的数据量，S3 Select 可以提高大多数经常访问 S3 数据的应用程序的性能，最高可提高 400%。

 

Offerslook全球率先使用该技术，并开创了多种使用方法（如基于Lambda+API gateway的使用方式）。目前，Offerslook业务中的点击日志以及部分转化数据查询都是基于S3功能构建，大幅度的提升转化数据和报表的稳定性。 在接下来的规划中，日志导出，BI，数据分析等业务也会以S3为中心构建整体的服务和功能模块。经在线验证和测试S3 select完全满足offerslook业务的特点，能够在S3海量数据中通过Key实时获取精准的一条数据内容，通过利用S3 99.99%的可用性保证offerslook业务系统高效稳定的运行。

 

作为全球第一家将此技术应用到生产环境中的企业，在会上，Offerslook不仅获得了业界的特别关注，也让众多与会人员特别好奇，纷纷前来请教和探讨。

 

在互联网+时代，创新技术层出不穷，企业的业务在不断变化，而企业的架构及运维也是随之发生改变。Offerslook一直坚持不断创新，力求为行业带来真正的改变，为用户解决真正的痛点。

 

关于Offerslook

Offerslook是全球领先的一站式效果营销管理云平台，致力于为移动广告网络、网盟提供线上广告追踪、监控、分析以及优化服务。通过运用云计算、大数据、智能算法等技术手段，Offerslook不断进行产品以及服务革新，旨在帮助全球营销者提供更高效智能，性价比更高的广告平台。Offerslook创立于2015年，总部位于香港，研发、运营中心位于广州，并在印度设立了分公司。公司创始团队来自阿里巴巴、华为等国内外知名互联网公司，在海外产品管理、移动营销以及大数据平台管理等领域有着丰富的技术积累与经验。经过短短2年多的发展，Offerslook已向40多个国家输出技术和品牌，服务了全球超过3000多家客户。
